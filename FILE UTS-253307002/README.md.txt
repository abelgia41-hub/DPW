Penjelasan Project: Website Penerimaan Mahasiswa Baru (PMB) untuk UTS

Project ini adalah sebuah Landing Page Pendaftaran Mahasiswa Baru untuk Stanford University (Tahun Ajaran 2026/2027), yang berfokus dari project ini adalah menyediakan platform yang informatif dan user-friendly bagi calon mahasiswa, supaya mereka bisa mengetahui jadwal, prosedur, hingga mengisi formulir pendaftaran secara digital.

website ini terdiri dari 3 bagian
Beranda (Halaman Utama): Diberanda terdapat informasi penting berupa Tabel Jadwal Pelaksanaan, dengan begitu calon mahasiswa bisa langsung mengetahui kapan masa pendaftaran dimulai sampai kapan pengumuman hasil seleksi keluar.
Alur Pendaftaran: Halaman ini berisi panduan langkah-demi-langkah (infografis). Tujuannya supaya calon pendaftar tidak bingung mengenai prosedur teknis pendaftaran, mulai dari registrasi akun sampai verifikasi berkas. note untuk ilustrasi saya ambil dari google dan saya edit sedikit dengan canva.
Isi Formulir (Biodata): Ini adalah bagian inti untuk pengumpulan data. Saya membuat form input yang mencakup Nama Lengkap, NIK (16 digit), dan pilihan Jalur Pendaftaran. saya mendesain bagian ini dengan sederhana, agar bisa memudahkan mahasiswa baru yang ingin menginputkan datanya.

Struktur yang Digunakan
HTML Ini adalah pondasinya. Saya pakai HTML untuk membuat struktur dasarnya, seperti menentukan posisi menu, posisi tabel jadwal, dan kotak untuk menginputkan biodatanya.
CSS digunakan supaya websitenya lebih berwarna dan menarik dilihat, Semua yang saya desain, saya menyatukannya di folder asset/CSS. Saya pilih kombinasi warna merah maroon dan putih supaya auranya persis seperti kampus Stanford.
JavaScript digunakan supaya websitenya bisa keliahatan lebih interaktif, file JavaScript (filenya ada di folder asset/javascript)
Manajemen File digunakan untuk memasukkan gambar dan ikon saya masukkan ke satu folder khusus namanya img, dengan begitu tidak ada file yang berantakan dan bisa dengan mudah diubah atau ditambahkan gambar atau ikonnya.
